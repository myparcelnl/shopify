Manual Shopify-plugin


Introduction

Using this Shopify plug-in, you can create five types of MyParcel shipments

  • Parcels within the Netherlands (home address and PostNL collection locations)
  • Parcels within the EU and outside of the EU
  • Letterbox parcel
  • Digital stamp
  • Unfranked label

Plugins van derden die we ondersteunen

  • PostNL Pick-up points

Let op: Wij leveren geen support op deze plugin!

1. Installation

1.1 New installation

Take the following steps to install the MyParcel plug-in:

  1. Go to Apps in your Shopify environment.
  2. Click Go to the Shopify Store in the top right-hand corner.
  3. Select the correct shop.
  4. Type MyParcel into the search bar and click Enter.
  5. Click the MyParcel NL app and then Add app.

1.2 Updating an existing installation

The app is updated automatically when changes are made. You do not need to update it yourself.

1.3 Testing

You can add Collect from a PostNL location as a delivery option at the checkout. Once you have installed and configured the plug-in, we recommend that you go through the order process and place a test order. This allows you to check that the checkout has the correct settings. You can also export shipments once the plug-in has been configured and test label downloads. Testing is free of charge, as long as you don't offer the labels to PostNL.
NB: you will be charged for digital stamps straight away.

2. Configuration

Go to Apps > Installed apps > MyParcel NL to open the MyParcel plug-in. Click Settings in the top left-hand corner to configure the plug-in.

2.1 Account

API key

The API key is the most important aspect of the link. You can find it in your MyParcel backoffice. In the top right-hand corner, click the symbol Shop settings > Select the correct shop > General > API settings.

2.2 General settings

Standard page selection

Here, you can specify whether you want to print the label on a standard A4 printer or on a label printer (A6).

Description on label

This options allows you to add a description to the shipment. The description will be printed in the top left-hand corner of the label. You can use it to search for the shipment in the MyParcel shipments summary. Use {order_name} to specify the order title or {order_number} to specify the order the number.

Fulfilment

You can give your orders Fulfilled status automatically once PostNL scans your shipment. If you would prefer to do this manually, select the Manual option. Go to Orders, select the relevant orders and click Fulfil orders above the order summary to change this status.

2.3 Standard export settings

Using the standard export settings, you can automatically create a parcel type, such as a parcel or a letterbox parcel, based on the shipping method selected by your customer. The shipping methods you have created will appear here so you can start setting your export preferences straight away. If you have not yet created any shipping methods, find out how to do this in the Shopify manual.

When you click a shipping method, a pop-up will appear. In the pop-up, you can specify whether the shipment can be sent through MyParcel, the shipment and parcel type that need to be used and whether extra options need to be added.

Nederland
Available shipment types: Home delivery and pick-up location
Available parcel types: Letterbox parcel, parcel, unfranked label and digital stamp
Available options: Signature upon receipt, deliver to recipient only, no-answer return, XL size, 18+ age check and insurance up to €5,000

PostNL Locaties
When you want to offer your customers the option to select a PostNL location for delivery, you must create another shipping method in Shopfiy. This new shipping method can then be linked in the default export settings.

  1. Go to “Apps" in Shopify
  2. Go to the MyParcel App
  3. In the MyParcel App click on “Settings” in the top left of the page.
  4. Scroll down true the settings until you find the “Default export settings”.
  5. Select the shopping methode you created for the PostNL locations.
  6. Turn on “Ship with MyParcel” and set the “Shipment type” to “PostNL pick-up”.
  7. After changing the settings click on “Save”.
When you have saved the settings, your customers will be able to use that shipping method to select en pick-up location. If a customers forgets to select one, they will receive a reminder email after about 30 minutes.

2.4 Herinneringsmail afhaallocatie

The MyParcel plug-in does not display the pick-up locations at the checkout. Your customer can select the preferred location on the thank-you page once they have placed their order. Switch on the option to send your customer a reminder email automatically after 30 minutes if they have forgotten to select a pick-up location.

2.5 World shipments

The HS code, weight per parcel and country of origin are required fields on customs forms for world shipments. Here, you can enter the standard values, which will be used if you have not entered values for the individual products.

NB: if you are sending more than one product, you need to enter a separate HS code for each product. If you do not do this, the parcel might not be cleared by the customs authorities.

2.6 Frequently Asked Questions about the configuration

Where can I enter the HS code and county of origin for a specific product?

Go to the Products page and select the product. You will find the form below at the bottom of the page. If you use ‘variants’, the form below will appear once you have clicked the relevant variant within the product type.

3. Use of plugin

3.1 Creating one shipping label

When opening an order, you will see the More actions button at the top of the page. In the dropdown menu, select the Print label option. You will be forwarded to the MyParcel app, where you can download the label straight away.

3.2 Aanmaken meerdere verzendlabels

If you want to download multiple orders at the same time, you can select them in the order summary. Then click the More actions button at the top of the page. In the dropdown menu, select the Print label option. You will be forwarded to the MyParcel app, where you can download the labels straight away.

3.3 Making changes to one shipment

If you want to export an order as a different parcel type than specified in the standard export settings, you can do this in the MyParcel app. Go to Apps > MyParcel NL and open the order. Here, you can create an export with an adjusted parcel type and add or delete any shipping options.

3.4 Pick-up location checkout

If you have linked the ‘pick-up location’ option to a shipping method (see chapter 2.3), your customers will be able to select a pick-up location on the thank-you page that appears after they have paid. To send your customers a reminder if they’ve forgotten to select a pick-up location, switch on the ‘Pick-up location reminder email’ (see chapter 2.4).

3.5 Emailing a return label

If your customer wants to return an order, you need to send them the return label by email. To do this, simply open the order and click the ‘More actions’ button at the top of the page. The ‘Send return label’ option will appear at the bottom of the dropdown menu. Select this option to send an email to your customer. If you want to change settings involving the email layout or excess to be paid by your customer, you can do this in your MyParcel backoffice. In the top right-hand corner, click the symbol Shop settings > Select the correct shop > Return.

3.6 FAQs about the plug-in

Why can my customer only select a pick-up location once they have paid?

In order to guarantee the security of payments, Shopify does not allow parties to add code to a checkout. However, we were still able to add this option to the plug-in after the checkout section. Sending your customers a reminder email is the most effective way to let your customers select a pick-up location (see chapter 2.4).

I have installed the plug-in but I receive an error when I try to export a shipment.

When configuring the plug-in, you might not have linked the active shipping methods to parcel types. Follow the steps in chapter 2.3 to set this up.
